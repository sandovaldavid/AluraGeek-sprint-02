## 💻 Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/sandovaldavid/AluraGeek-sprint-02.git
    ```

2. Navigate to project directory:

    ```bash
    cd AluraGeek-sprint-02
    ```

3. Open `index.html` in your browser or use a local server
